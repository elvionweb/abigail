# Anett's Birthday Website 💕

Luxury romantic, fully responsive birthday single-page website built with React + Vite, Tailwind CSS, Framer Motion, and Howler.js.

## Setup

```bash
npm install
npm run dev
npm run build
```

## Replace Photos

Drop JPEG files named `ank.jpeg` through `ank30.jpeg` into `/public/photos/`.

## Replace Music

Drop MP3 files into `/public/music/` using the exact filenames referenced in `SITE_CONFIG.playlist` inside `/src/config.js`.

## Replace Voice Message

Place `voice-message.mp3` in `/public/voice/`.

## Edit Personal Content

Edit only `/src/config.js` for names, captions, letters, timeline, playlist, and share text.

## Deploy to Vercel

1. Push project to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import the repository
4. Deploy
